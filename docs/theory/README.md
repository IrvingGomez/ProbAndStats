# How Theory Connects to the Application

Each major section of the application corresponds to theoretical foundations:

| Application Area        | Theoretical Foundation |
|------------------------|------------------------|
| Descriptive Statistics | Moments, distribution summaries |
| Graphical Analysis     | Empirical distribution, density estimation |
| Confidence Intervals   | Sampling distributions, asymptotic theory |
| Prediction Intervals   | Model uncertainty and variance decomposition |
| Hypothesis Testing     | Test statistics and significance levels |
| Linear Regression      | Least squares and statistical inference |

Students are encouraged to:

- Review the relevant theory before using a method.
- Compare theoretical formulas with implementation in `core/`.
- Reflect on assumptions behind each estimator or test.

# Educational Philosophy

The Himmapan Lab ecosystem emphasizes:

- Mathematical understanding
- Transparent implementation
- Reproducible computation
- Critical interpretation of results

This project is structured so that students can:

1. Learn the theory.
2. Inspect the implementation.
3. Experiment with data.
4. Extend the methodology.
